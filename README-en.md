## 📖 What is OpenClaw?

OpenClaw is a local AI assistant framework that helps you:

- 🖥️ **Automation** — Browser control, file management, code execution
- 💬 **Multi-channel** — Telegram, Discord, 飞书, WhatsApp
- 🧠 **Memory System** — Long-term memory + daily notes
- 🔌 **Skills Ecosystem** — Extensible plugins

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🏠 **Local First** | Data stays local, privacy protected |
| 🎯 **Fine-grained Control** | Granular permissions, secure |
| 🔄 **Continuous Learning** | Memory system makes AI understand you |
| 🛠️ **Extensible** | Plugin architecture |

## 🚀 Quick Start

```bash
# Install
npm install -g openclaw

# Initialize
openclaw init

# Start
openclaw start
```

## 🛠️ Tech Stack

- **Runtime**: Node.js
- **AI**: OpenAI / Anthropic / Ollama / MiniMax
- **Channels**: Telegram, Discord, 飞书, WhatsApp, Signal
