## 📖 什么是 OpenClaw？

OpenClaw 是一个本地运行的 AI 助手框架，可以帮你完成：

- 🖥️ **自动化工作流** — 浏览器操作、文件管理、代码执行
- 💬 **多渠道接入** — Telegram、Discord、飞书、微信
- 🧠 **记忆系统** — 长期记忆 + 每日笔记，跨会话保持上下文
- 🔌 **技能生态** — 可扩展的技能市场，随装随用

## ✨ 特性

| 特性 | 说明 |

## 🚀 快速开始

```bash
# 安装
npm install -g openclaw

# 初始化
openclaw init

# 启动
openclaw start
```

详细步骤见 [安装指南](./docs/01-setup.md)

## 🛠️ 技术栈

- **运行时**: Node.js
- **AI 接入**: OpenAI / Anthropic / Ollama / MiniMax
- **渠道**: Telegram, Discord, 飞书, WhatsApp, Signal
- **存储**: 本地文件系统 + SQLite
