## 🚀 快速开始

```bash
# 1. 安装（30 秒）
npm install -g openclaw

# 2. 初始化（1 分钟）
openclaw init

# 3. 启动
openclaw start

# 4. 立即发送第一条指令：
# "帮我写一个 Python 脚本，输出 Hello World"
```

## 🛠️ 技术栈

- **运行时**: Node.js 18+
- **AI 模型**: OpenAI / Anthropic / Ollama / MiniMax
- **渠道**: Telegram, Discord, 飞书, WhatsApp, Signal, Webchat
- **存储**: 本地文件系统 + SQLite
